# ldap
Refer to:
- https://gopkg.in/ldap.v3

## Installation
Please make sure to initialize a Go module before installing core-go/ldap:

```shell
go get -u github.com/core-go/ldap
```

Import:

```go
import "github.com/core-go/ldap"
```

## Detail:
In go.mod file
```mod
require (
	gopkg.in/ldap.v3 v3.1.0
)
```
